# **App Name**: Frontend Flipper

## Core Features:

- Welcome Message: Display a welcome message using textual information.
- Theme Toggle: Button to toggle the UI between light and dark mode.
- Message Regeneration: AI tool to randomly regenerate the welcome message with a new message using generative AI.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to reflect trustworthiness and stability, contrasting with a light or dark background depending on the mode.
- Background color: Very light gray (#F0F0F0) for light mode, dark gray (#333333) for dark mode, both with low saturation.
- Accent color: A vibrant purple (#9C27B0), for the message regenerate call to action, providing contrast against the primary color and background.
- Font: 'PT Sans', a humanist sans-serif, for both headings and body, as it combines a modern look with approachability.
- Simple layout with a clear visual hierarchy: welcome text at the top, buttons arranged horizontally below.
